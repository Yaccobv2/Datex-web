# Datex-web
Web app to communicate with SensHat on RaspberryPI.

This is web page app on RaspberryPI's server created with lighttpd that can use every functionality of SensHat.

# Used Tools
* html
* css
* javascript
* bootstrap4
* python
* php
* json files
* jQuery
